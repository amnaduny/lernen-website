Core Program

Apa itu Core Program?

Core Program adalah program bimbingan intensif untuk mempersiapkan studi lanjut ke luar negeri (S1/S2), baik dengan maupun tanpa beasiswa. Program ini dirancang bagi siswa SMA dan mahasiswa S1 yang ingin melanjutkan studi dengan percaya diri melalui pemahaman menyeluruh mengenai proses, dokumen, dan seleksi.

Durasi program meliputi 12 sesi (paket basic) atau 23 sesi (paket premium dan advanced) selama 4 bulan. Metode bimbingan meliputi pembahasan persyaratan dasar, dokumen, latihan soal, diskusi, peer review, dan simulasi untuk memaksimalkan pemahaman mentee.
